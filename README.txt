Introduction
============

Dexterity based product for adding Publications to a site.  The fields were intended to be broad, so that citation formats can vary, depending on the discipline.  

Usage Instructions
========================

You must first add a Publications Folder to your site.  From Add New, select Publications Folder.  From there, you will then be permitted to add a new publication to the folder.  There is a default view for the folder as well as a simple Jquery based search.
